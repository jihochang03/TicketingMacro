# 티켓팅 자동화 스크립트

티켓 예매 사이트에서 티켓을 자동으로 예약하는 매크로입니다. Selenium을 통해 브라우저 자동화를 구현하고, Tkinter로 사용자 입력을 수집하며, PyAutoGUI를 사용하여 화면과 상호작용할 수 있습니다. 스크립트에는 각 극장별로 정의된 조건에 따라 특정 좌석을 선택하는 로직이 포함되어 있습니다.

## 사전 준비

1. **Python**: Python이 설치되어 있어야 합니다 (Python 3.7+ 권장).
2. **필수 라이브러리**: 다음 명령어로 필요한 라이브러리를 설치하세요.
   ```bash
   pip install selenium keyboard pyautogui pillow
   ```
3. **WebDriver**: Chrome WebDriver가 필요합니다. [ChromeDriver](https://sites.google.com/chromium.org/driver/)에서 현재 사용하는 Chrome 버전에 맞는 드라이버를 다운로드하고 시스템 PATH에 추가하세요.

## 스크립트 개요

이 스크립트는 다음 작업을 자동으로 수행합니다.

1. **브라우저 초기화**: Chrome을 열고 티켓 예매 사이트로 이동합니다.
2. **사용자 입력**: Tkinter GUI를 통해 필요한 정보(날짜, 시간, 극장명 등)를 수집합니다.
3. **자동화 동작**:
   - 설정된 티켓팅 시간이 될 때까지 대기합니다.
   - 사용자 정의 조건에 따라 좌석을 검색하고 선택합니다.
   - 극장별 좌석 선택 로직을 처리합니다.
4. **확인 및 완료**: 예약 확인 메시지를 검사하고, 좌석이 없으면 새로고침하여 재시도합니다.

## 사용법

1. **스크립트 실행**:
   ```bash
   python script_name.py
   ```

2. **정보 입력**:
   - Tkinter 팝업 창에서 요구되는 정보를 입력합니다 (날짜, 시간 등).
   - 모든 정보를 입력한 후 “Submit” 버튼을 클릭합니다.

3. **좌석 선택**:
   - 스크립트는 목표 시간까지 기다린 후, 설정된 조건에 따라 사용할 수 있는 좌석을 검색합니다.
   - 일치하는 좌석을 찾으면 스크립트가 해당 좌석을 클릭하여 예약을 시도합니다.

4. **확인**:
   - 좌석을 선택한 후, 스크립트는 예약 성공을 확인하는 메시지를 검사하여 예약이 완료되었는지 확인합니다.
   - 메시지를 찾지 못하면 페이지를 새로고침하여 재시도합니다.

## 커스터마이징

### 극장 좌석 선택 조건
각 극장은 좌석을 식별하고 선택하는 고유 조건과 XPath를 가지고 있습니다. 이를 변경하려면 `select_and_click_seat()` 함수 내 조건을 수정하세요.

### 새로운 극장 추가
새로운 극장을 추가하려면 `select_and_click_seat()` 함수 내 `elif` 블록을 추가하고, 해당 극장의 좌석 선택 조건과 XPath 위치를 지정하세요.

### PyAutoGUI 클릭 좌표 조정
PyAutoGUI의 클릭 동작에 사용되는 좌표 (`x_coord` 및 `y_coord`)를 조정하려면 `while` 루프 내 값을 수정하세요.

## 제한 사항

- **정적 XPath 사용**: 웹사이트가 변경되면 요소를 찾기 위한 XPath도 업데이트해야 할 수 있습니다.
- **화면 해상도**: 특정 해상도에 맞춰져 있으므로, 필요한 경우 창 크기 조정이나 PyAutoGUI 좌표를 변경하세요.
- **브라우저 호환성**: 이 스크립트는 Chrome에 최적화되어 있습니다. 다른 브라우저를 사용하려면 WebDriver와 관련 설정을 업데이트해야 합니다.

## 문제 해결

1. **요소를 찾지 못함**: 요소를 찾을 수 없는 경우 XPath가 올바른지 확인하세요. 웹사이트를 점검하여 XPath를 조정해야 할 수 있습니다.
2. **WebDriver 오류**: Chrome과 Chrome WebDriver 버전이 일치하는지 확인하세요.
3. **타임아웃 문제**: 스크립트 실행 속도가 너무 빠르거나 느린 경우 `WebDriverWait` 및 `time.sleep`과 같은 대기 시간을 조정하세요.
